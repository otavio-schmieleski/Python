Transformar uma arquivo python em executavel utiliza a biblioteca Auto-py-to-exe

1º instala no terminal com acesso administrador
  . pip install auto-py-to-exe

2º executa a linha de comando a baixo para abrir a janela
  . auto-py-to-exe

3º preenche as informações e clica no botão converter.

link para documentação https://pypi.org/project/auto-py-to-exe/

link para tutorial: https://www.youtube.com/watch?v=Kp_41haOVQk
